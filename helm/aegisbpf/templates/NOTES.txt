=== aegis-next DaemonSet ===

The next-gen BPF arena agent is deployed. It targets nodes labeled:
  aegisbpf.io/arena-capable=true

Check DaemonSet status:
  kubectl get ds {{ include "aegisbpf.fullname" . }}-next -n {{ .Release.Namespace }}

View agent logs:
  kubectl logs -l aegisbpf.io/variant=next -n {{ .Release.Namespace }} --tail=50

{{- if .Values.operator.enabled }}
Policies are managed via AegisPolicy CRDs. Apply a policy:
  kubectl apply -f - <<EOF
  apiVersion: aegisbpf.io/v1alpha1
  kind: AegisPolicy
  metadata:
    name: block-miners
  spec:
    mode: enforce
    execRules:
      denyComm: [xmrig, minerd]
  EOF

View merged policy ConfigMap:
  kubectl get cm aegis-merged-policy -n {{ .Release.Namespace }} -o yaml
{{- else if .Values.policy.enabled }}
Static policy is mounted from ConfigMap {{ include "aegisbpf.fullname" . }}-policy.
{{- end }}
{{- end }}
